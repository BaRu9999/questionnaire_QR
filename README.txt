口コミ・アンケートQRシステム 一式

【内容】
- index.html
  店頭QRから開くアンケートページ本体です。
- qr_generator.html
  GitHub Pagesで公開後のURLを入力して、QRコードを表示する簡易ページです。
- 設定メモ.txt
  差し替えるURLと作業手順のメモです。

【使い方】
1. index.html をGitHub Pagesにアップロードします。
2. index.html の上部にある3つのURLを差し替えます。

   GOOGLE_REVIEW_URL
   → Googleクチコミ投稿用リンク

   LINE_ADD_URL
   → LINE公式アカウントの友だち追加URL

   IMPROVEMENT_FORM_URL
   → 不満・改善要望を受けるGoogleフォームURL

3. GitHub Pagesで公開されたURLを qr_generator.html に入力します。
4. 表示されたQRコードを保存し、POPに配置します。

【導線】
QRコード
↓
満足度アンケート
↓
満足した
→ Googleクチコミを書く
→ LINE友だち追加で5%OFF
→ ご意見・ご要望フォーム

普通
→ ご意見・ご要望フォーム
→ Googleクチコミを書く
→ LINE友だち追加で5%OFF

不満があった
→ 改善フォーム
→ Googleクチコミを書く
→ LINE友だち追加で5%OFF

【注意】
Googleクチコミ対策として、満足していない方にもGoogleクチコミリンクは表示される設計にしています。
